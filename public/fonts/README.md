# 폰트 파일 안내

## Orbitron Bold 폰트 준비 방법

1. **Google Fonts에서 다운로드**
   - https://fonts.google.com/specimen/Orbitron
   - Orbitron Bold (700) 선택 후 다운로드

2. **Typeface.js로 변환**
   - 온라인 변환기: https://gero3.github.io/f3d/
   - 또는 npm 패키지: `npm install -g typeface-generator`
   - 명령어: `typeface-generator Orbitron-Bold.ttf`

3. **파일 저장**
   - 변환된 `orbitron_bold.typeface.json` 파일을 이 폴더에 저장
   - 경로: `/public/fonts/orbitron_bold.typeface.json`

## 대안: 온라인에서 직접 사용

온라인에서 호스팅된 Orbitron Bold typeface.json을 찾아서 URL로 사용할 수도 있습니다.
